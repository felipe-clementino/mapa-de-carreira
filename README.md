# Mapa de Carreira Back-end

## Sobre o Projeto

Este projeto apresenta meu **Mapa de Carreira Back-end**, desenvolvido com o objetivo de planejar e visualizar minha evolução profissional na área de tecnologia.

O mapa descreve as etapas da minha trajetória como desenvolvedor, destacando competências técnicas, habilidades comportamentais, tecnologias essenciais e objetivos de crescimento para cada nível de carreira.

---

## Objetivos

* Consolidar conhecimentos em desenvolvimento Back-end.
* Evoluir progressivamente entre os níveis Júnior, Pleno e Sênior.
* Desenvolver competências em arquitetura de software e sistemas escaláveis.
* Aprender práticas modernas de DevOps e computação em nuvem.
* Fortalecer habilidades técnicas e comportamentais necessárias para o mercado de trabalho.
